-------- conversor de monedas --------
implementar un conservor de monedas que se ejecute en la consola <Java>

pre requisitos:

1. Java Depelopment kit (JDK) VERSIÓN 17
2. Libreria Gson: nos permite convertir la respuesta JSON de la API en objetos de Java. (incluir el archivo Jar de Gson en classpath)
3. Clave API: utilizaremos una clave de ExchangeRate que nos permite obtener el valor de la conversión de forma inmediata y en linea


                  ------------Configuraciones------------

Configurar API:
1. Abrir el archivo main.java
2. Ingresar tu API Key de ExchangeRate en la siguiente linea de codigo:

   private static final String API_KEY = "TU_CLAVE_AQUI"; 

          ------------------------Menú----------------------
Para el menú utilizaremos siete conversiones:
1. Dolar(USD) a Libra Esterlina(GBP)
2. Peso Chileno(CLP) a Euro(EUR)
3. Peso Chileno(CLP) a Dolar(USD)
4. Peso Chileno(CLP) a Peso Argentino(ARS)
5. Dolar(USD) a Peso Chileno(CLP)
6. Dolar(USD) a Yuan Chino(CNY)
7. Yuan Chino(CNY) a Peso Chileno(CLP)
0. Salir
